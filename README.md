# ExoticCo  Business Website Template

## Overview
This project is a modern, responsive business website template built for
software companies, startups, and tech agencies. It is designed with clean
UI, scalable structure, and real-world client use in mind.

## Features
- Responsive design for all devices
- Clean and modern layout
- Easy to customize sections
- SEO-friendly structure
- Fast loading performance

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Use Cases
- Software development companies
- Startup landing pages
- Business portfolio websites

## Getting Started
1. Clone the repository
2. Open `index.html` in your browser
3. Customize content as needed
